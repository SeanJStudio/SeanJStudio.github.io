Dfuze

Dfuze is an offline desktop application designed for B2B data onboarding and ETL. See it in action here.

My job involves working with numerous clients who send database files in a variety of formats and inconsistant structures. This means that I am responsible for handling many new database manipulation requests, from segmenting charity donors to identifying and removing records based on name and address comparisons. Some of these tasks can be repetitive or time-consuming, while others require solutions beyond traditional SQL queries.

To address these challenges, I developed Dfuze, which enables me and my colleagues - regardless of our backgrounds or experiences - to easily accomplish our daily job duties.

Some of the key features of Dfuze include the ability to combine multiple data files (even of different file types or tables) into one database, deduplicate names and addresses quickly and accurately, and regex pattern search an entire database while highlighting matched parts. Dfuze also maps data to required fields automatically using string matching algorithms or saves selected ones for future use, exports data to various formats, splits files based on data values, merges and normalizes disparate data sources, and automates tasks with programming templates.
