PDFUtil

I work with a lot of PDFs which made me develop this utility application. I wrote it in Java on top of the Apache PDFBox library. The coolest feature is the ability to extract text that is embedded or via Tess OCR. The video example above shows how you can preview a PDF file and draw an area around the table you wish to extract to comma delimited data.

Theres lots of ways it can manipulate PDFs but some of my favorite things are:

  1. Extracting names and address blocks normalized into their separate components to comma delimited data.
  2. Extracting tables to comma delimited data.
  3. Find and replace text with Regex Patterns (way harder to do than I thought).
  4. Interweave multiple PDFs and their pages together.
  5. Decrypt a PDF.
