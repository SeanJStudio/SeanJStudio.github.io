Proppy

Proppy is a GUI-less scraping application I developed in Java, HTMLUnit and SQL Server to track Canadian properties for sale everyday. By using a combination of rotating residential proxies, rotating user agents and randomization I am able to scrape a couple of the most popular property sites in Canada. Every day it automatically scrapes sites between random times for all the new properties and even accounts for daylight savings time so it can run unattended.

In order to properly track the properties it corrects addresses to Canada Post SERP standards and checks if the property has already been added within a 3 month period before adding it. These sites did not provide properties for Manitoba and just had a few for Quebec. I'm assuming this is because they pay for each realtor groups property listings and it doesn't make sense to buy them based on their target market.
