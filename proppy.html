Proppy

Proppy is a unique application I developed using Java, HTMLUnit, and SQL Server. Its purpose is to track properties for sale in Canada on a daily basis.

By utilizing a combination of rotating residential proxies, rotating user agents, and randomization, Proppy is able to scrape some of the most popular property sites in Canada. It scrapes sites between random times every day for all the new properties and even accounts for daylight savings time, allowing it to run unattended.

To ensure accurate tracking, Proppy corrects addresses to Canada Post SERP standards and checks if the property has already been added within a 3-month period before adding it. Although some sites did not provide properties for Manitoba and only had a few for Quebec, I suspect it's because they pay for each realtor group's property listings and it doesn't make sense to buy them based on their target market.
