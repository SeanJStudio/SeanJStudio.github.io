Tagd

Tagd is a user-friendly application that I created using Java and Apache PDFBox. It is designed to provide a solution for quality control forms in printing processes.

Tagd allows printers to easily check the quality of a print run at regular intervals, such as every 500 pieces. The application generates a PDF document that contains 4 forms imposed on a letter sized sheet in cut and stack order. This feature reduces paper waste and helps to keep the forms organized.

One of the standout features of Tagd is its customer database. The application holds a database of customers, and it can suggest them to users as they type. This feature not only saves time for the user but also ensures accuracy in the printing process.
